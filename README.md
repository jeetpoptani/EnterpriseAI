# AI Media Describer

## What It Will Be

This project is an AI-powered media description service that analyzes uploaded images and videos and returns accurate natural-language descriptions. It is designed to provide frame-aware visual understanding for videos and concise scene-level summaries for both image and video content.

## Tech Stack

- Python
- FastAPI
- Groq API
- Uvicorn
- FFmpeg
- Pydantic
